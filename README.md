Pin 7  (AVSS)     →      GND<br>

Pin 5  (DRDY)     →      Any GPIO (for data ready interrupt)<br>
Pin 4  (SA0)      →      GND or 3.3V (I2C address bit 0)<br>
Pin 2  (SA1)      →      GND or 3.3V (I2C address bit 1)<Br>
